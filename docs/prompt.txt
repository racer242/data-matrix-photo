Напиши веб-приложение для сканирования Data Matrix кода, изображенного на фото.
Фото может содержать изображение кода в любом размере, любой позиции, любом освещении, под любым наклоном.
Приложение должно работать на JS
Приложение должно быть изолировано, чтобы избежать конфликта с другими элементами, скриптами и стилями.
Приложение должно быть представлено в виде JS-скрипта, который можно вставить в HTML-документ.
Приложение должно работать на всех современных браузерах.
Реализация приложения должна быть минимальной, лаконичной, простой в интеграции и использовании.

Результатом создания приложения, должны быть следующие артефакты:
- JS-скрипт библиотеки приложения
- HTML-страница с примером использования (отображение загружаемого фото, кнопка "Загрузить", кнопка "Считать", вывод результата считывания или ошибки)
- readme.md - описание приложения, инструкция по интеграции и использованию.

JS интерфейс, который должно поддерживать приложение:
dataMatrixPhotoReader.upload() - загружает фото в клиент
dataMatrixPhotoReader.start() - стартует сканирование фото
dataMatrixPhotoReader.onUpload() - обработчик готовности загрузки фото
dataMatrixPhotoReader.onReady() - обработчик готовности данных - возвращает распознанный объект.
dataMatrixPhotoReader.onUploadError() - обработчик ошибки загрузки кода - возвращает объект с ошибкой.
dataMatrixPhotoReader.onReadError() - обработчик ошибки считывания кода - возвращает объект с ошибкой.
